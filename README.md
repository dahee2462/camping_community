# ⛺ Camping Community :: 캠핑 커뮤니티 ⛺
캠핑에 대한 정보제공을 위한 JAVA기반 캠핑 커뮤니티 사이트입니다.

## ⏱️ 개발기간
* 2023/12/28 - 2024/01/19

## 🧑‍🤝‍🧑 팀원소개
|이름|담당기능|
|------|---|
|**이준택 (팀장)**|로그인, 회원가입, 아이디/비밀번호 찾기, 날씨 API, 캠핑지역 게시글 CRUD|
|**전용휘**|캠핑장비 CRUD, (대)댓글, 좋아요 기능, 게시판 상세보기, 게시판 작성및 수정,|
|**정다희**|메인페이지, 출석체크 CRUD, Q&A CRUD, 게시판 상세보기, 게시판 작성및 수정, 웹디자인, PPT|
